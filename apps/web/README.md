# WebVitals Web Dashboard

This is the Next.js 15 frontend dashboard application for WebVitals.io.

## Development

```bash
# From the monorepo root
npm run dev

# Or from this directory
npm run dev
```

## Build

```bash
# From the monorepo root
npm run build

# Or from this directory
npm run build
```

## Dependencies

This app depends on:
- `@webvitals/types` - Shared TypeScript types
- `@webvitals/utils` - Shared utility functions

## Structure

- `app/` - Next.js App Router pages and components
- `lib/` - Application-specific utilities and configuration
- `scripts/` - Build and development scripts
